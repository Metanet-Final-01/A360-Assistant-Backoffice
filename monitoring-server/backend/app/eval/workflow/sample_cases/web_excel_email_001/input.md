# 업무정의서: 웹 시세 조회 후 엑셀 저장 및 메일 발송

매일 오전 9시에 담당자는 웹 페이지에서 금 시세를 조회한다.
최근 3일치 시세만 확인하고, 날짜와 시세를 엑셀 파일에 저장한다.
엑셀 파일은 사용자가 지정한 폴더에 저장한다.
저장이 끝나면 담당자에게 결과 파일을 첨부하여 이메일로 발송한다.
조회 결과가 없으면 엑셀을 만들지 않고 오류 안내 메일을 발송한다.

## 기대 의미 단위

- 웹 페이지 접속
- 최근 3일치 데이터 조회 또는 반복 처리
- 엑셀 파일 생성 또는 열기
- 날짜/시세 데이터 쓰기
- 파일 저장
- 이메일 발송
- 결과 없음 예외 분기
